# SQL Queries Portfolio

This repository is my SQL portfolio.
It contains queries I wrote for different tasks to demonstrate my SQL style and skill level.

The project structure is intentionally simple and not strictly formalized: the main goal is to showcase the queries themselves.

## Files

### `Movie_database_analysis.sql`
- A set of queries for a movie rental training database.
- Each query is preceded by a comment in the file describing the task.
- Includes tasks using `JOIN`, `CTE`, window functions, aggregations, and filtering.

### `User_activity_stats.sql`
- A query for analyzing weekly user login activity from the start of the application usage.
- Calculates:
  - total number of logins per week;
  - number of unique users logging in per week;
  - number of new users who logged in for the first time in each week.
